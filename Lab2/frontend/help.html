📚 Как пользоваться приложением

📤 Загрузка документа

  1. Нажмите "Выберите файл" и выберите документ (.txt, .rtf, .pdf, .docx, .doc)
  2. (Опционально) Введите метаданные в формате JSON
  3. Нажмите "Загрузить"

🔎 Поиск по корпусу

  1. Введите запрос в поле поиска
  2. Выберите тип поиска:
    • По лемме — поиск по словарной форме (бежать → бежал, бегу)
    • По словоформе — точное совпадение слова
    • По части речи — фильтр по грамматической категории
    • По регулярному выражению — расширенный поиск по шаблону
  3. Настройте контекст и лимит результатов
  4. Нажмите "Найти"

📁 Управление документами

  • 📥 Скачать — получить оригинальный файл
  • 📊 Статистика — просмотреть аналитику документа
  • ✏️ Редактировать — изменить имя и метаданные
  • 🗑️ Удалить — удалить документ из корпуса

📦 Экспорт корпуса

Нажмите "Экспорт (ZIP)" для скачивания архива со всеми документами и метаданными.

❓ Глоссарий

Лемма
Начальная (словарная) форма слова. Например: "бежать" для "бежал", "бегу", "бежит".
Конкорданс
Список вхождений слова с контекстом — несколько слов до и после искомой единицы.
POS (Part-of-Speech)
Часть речи. Основные категории:
  • NOUN — Существительное (дом, свобода)
  • VERB — Глагол (читать, был)
  • ADJ — Прилагательное (красный, быстрый)
  • ADV — Наречие (быстро, вчера)
  • PRON — Местоимение (я, он, этот)
  • ADP — Предлог (в, на, под)
  • CONJ — Союз (и, но, что)
  • PART — Частица (не, бы, ли)
  • INTJ — Междометие (ох, эй)
  • NUM — Числительное (пять, первый)
  • AUX — Вспомогательный глагол (быть в составе сказуемого)
Токен
Минимальная единица текста после токенизации (слово, знак препинания).
Метаданные
Дополнительная информация о документе: автор, год, жанр, источник и т.п. в формате JSON.
